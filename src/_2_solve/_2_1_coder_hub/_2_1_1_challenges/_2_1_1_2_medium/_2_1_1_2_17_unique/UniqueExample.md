# <div dir="rtl">العثور على العنصر الفريد في المصفوفة</div>

## <div dir="rtl">السؤال</div>

<ul dir="rtl">
<li>
<b>
قم بكتابة دالة function تستقبل مصفوفة من نوع integer تقوم الدالةـ function بالبحث عن الأرقام الفريدة في المصفوفة وثم تقوم بإرجاعها داخل مصفوفة جديدة.
</b>
</li>
</ul>

---

## <div dir="rtl">المخرجات المتوقعة</div>

### <div dir="rtl">اختبار 1</div>

#### <div dir="rtl">المُدخلات</div>

```text
arr = [1, 1, 1, 2, 1, 1]
```

#### <div dir="rtl">المُخرجات</div>

```text
[2]
```

### <div dir="rtl">اختبار 2</div>

#### <div dir="rtl">المُدخلات</div>

```text
arr = [3, -4, 3, 3, 3]
```

#### <div dir="rtl">المُخرجات</div>

```text
[-4]
```

### <div dir="rtl">اختبار 3</div>

#### <div dir="rtl">المُدخلات</div>

```text
arr = [2, 4, -2]
```

#### <div dir="rtl">المُخرجات</div>

```text
[2,4,-2]
```

### <div dir="rtl">اختبار 4</div>

#### <div dir="rtl">المُدخلات</div>

```text
arr = [2, 3, 2, 6, 2]
```

#### <div dir="rtl">المُخرجات</div>

```text
[3,6]
```
