# <div dir="rtl">عدد عناصر المصفوفة</div>

## <div dir="rtl">السؤال</div>

<ul dir="rtl">
<li>
<b>
قم بكتابة function تستقبل مصفوفة array مكونة من أعداد صحيحة من نوع integer، ثم تقوم الدالة بإرجاع عدد عناصر لتلك المصفوفة.
</b>
</li>
</ul>

```

```
